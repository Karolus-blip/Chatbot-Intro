# Chatbot-Intro

Proyecto inicial de diseño conversacional enfocado en la simulación de un asistente virtual para una librería.

## Contenido del repositorio

- Diálogo para devolución de libros
- Diagrama de flujo conversacional
- Prototipo lógico en Python
- Interfaz básica en HTML

## Objetivo

Explorar estructuras de interacción entre usuarios y asistentes virtuales en contextos culturales y de atención al cliente.

## Estado del proyecto

Primera fase de desarrollo.
